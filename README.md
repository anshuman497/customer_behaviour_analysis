üîç Steps and Workflow

1. Data Loading

Imported dataset into Python using Pandas.

Verified encoding and column consistency.


2. Data Cleaning

Handled missing values and outliers.

Standardized categorical values (e.g., payment methods, product categories).

Converted dates to proper datetime format.


3. Exploratory Data Analysis (EDA)

Generated summary statistics and visual insights.

Used correlation heatmaps, histograms, and bar charts for variable relationships.

Derived metrics such as Average Order Value (AOV), purchase frequency, and top categories.


4. SQL Integration

Uploaded cleaned data to PostgreSQL/MySQL/SQL Server.

Ran analytical SQL queries such as:

Top-spending customers

Most popular product categories

Revenue by month and region

Payment preferences by gender and age



5. Power BI Dashboard Creation

Connected database or CSV to Power BI.

Designed an interactive dashboard with:

KPIs: Total Revenue, Average Bill, Customer Count

Filters: Category, Payment Method, Gender, Month

Charts: Sales trends, category distribution, and demographic insights



6. Reporting and Presentation

Summarized findings in a Gamma presentation highlighting:

Key insights and trends

Actionable business recommendations

Visual snapshots of Power BI dashboard
